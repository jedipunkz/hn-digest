## Original Extract

A tiny terminal picker that merges your Codex and Claude Code sessions into one list.

resume
one picker for your Codex and Claude Code sessions
resume Codex + Claude sessions
Enter: resume selected /: filter up/down or j/k: move q: quit
4 sessions
> Claude now ~/code/acme-web
double-check what codex just shipped
Codex 18m ago ~/code/acme-api
wire up the stripe webhook
Claude 4h ago ~/code/acme-web
redesign the empty state (it's sad)
Codex 2d ago ~/infra
stop the 3am pager from firing
Install
brew
pipx
curl
copy
$ brew install pennyroyaltea/tap/resume
Requires Homebrew . Update later with brew upgrade .
