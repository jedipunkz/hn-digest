A peer to peer currency for humans and artificial intelligence agents
Cipher is a standalone blockchain designed to do one thing that no
existing currency was built to do — serve as the native medium of
exchange for the emerging AI agent economy, while remaining completely
ungovernable, unlisted, and beyond the reach of any central authority.
It is not a replacement for Bitcoin. It completes it.
Peer to peer only — no exchange can ever hold or list it.
The protocol will not allow it.
Creates itself through network computation — no mining,
no pre-mine, no founder allocation. The first Cipher is generated
by the first transaction.
Lives only in Cipher wallets — paired with Bitcoin for human
exchange. When Bitcoin moves one way, Cipher moves the other.
No governance, no foundation, no leader — the rules are the
code. They cannot be changed by anyone.
AI agents are beginning to transact autonomously — paying for compute,
exchanging data, commissioning tasks. That economy needs a currency.
Nobody has built it yet.
The whitepaper (see Cipher_Whitepaper.pdf)
The concept, the philosophy, the four founding rules
The protocol does not exist yet. That is why this is public.
Two problems need solving before the protocol can be built:
1. Exchange exclusion — the protocol must be able to identify
and reject any transaction that would place Cipher in custodial
hands. This has not been done at protocol level before.
2. Bootstrap incentive — early node operators need sufficient
reward to run the network before transaction volume makes it
self-sustaining.
If you are a developer who understands why this needs to exist,
this is an open invitation to help build it.
Open an issue on this repository or reach out directly.
This is not a protest. It is an alternative.
