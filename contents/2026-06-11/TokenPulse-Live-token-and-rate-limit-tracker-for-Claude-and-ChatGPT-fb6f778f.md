- コンテキスト ウィンドウ — 使用されたトークン、残り、パーセンテージ
- クロードのレート制限 — クロードの API からの実際の 5 時間と 7 日間の使用量 (カウントダウンがリセットされる)
- 毎日の使用履歴 — 1 日あたりのピークトークン使用量の自動追跡
- スマート通知 — 75%、90%、100% でアラート
Claude と ChatGPT の両方で動作します (1 つの拡張機能、2 つのプラットフォーム)。
プライバシー: すべてのデータはデバイス上に残ります。サーバーもアカウントも追跡もありません。

Never hit a token limit unexpectedly again.
TokenPulse tracks your context window and rate limits in real time — directly inside ChatGPT and Claude. No switching tabs, no guessing, no surprises.
A live token bar sits above the input box. As you chat, it counts tokens and shows exactly how much context window you have left.
Click the toolbar icon for the full dashboard:
- Context Window — tokens used, remaining, and percentage
- Claude Rate Limits — real 5-hour and 7-day usage from Claude's API with reset countdowns
- Daily Usage History — automatic tracking of peak token usage per day
- Smart Notifications — alerts at 75%, 90%, and 100%
Works on both Claude and ChatGPT — one extension, two platforms.
