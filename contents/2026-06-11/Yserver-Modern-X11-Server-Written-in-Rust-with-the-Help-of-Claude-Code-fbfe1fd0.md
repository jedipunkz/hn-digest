Open-source developer Jos Dehaes wrote in to Phoronix today in announcing a new X11 server he has been working on from scratch that has been quietly developed to this point but now ready to announce to the world..

Categories
Computers Display Drivers Graphics Cards Linux Gaming Memory Motherboards Processors Software Storage Operating Systems Peripherals
YSERVER: Modern X11 Server Written In Rust With The Help Of Claude Code
"The goal is not to clone Xorg. It is to provide a practical X11 server that runs real desktop environments, window managers, and applications on modern Linux while dropping legacy baggage (multiple screens, non-TrueColor visuals, indirect GLX, the DDX driver ABI, endian-swapped clients, and so on)."
