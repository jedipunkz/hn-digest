Anthropic の新しいエージェント SDK の価格設定は Codex にとって有利です
Claude サブスクリプションは開発者向けに設計されています
多くの開発者は、エージェントと一緒に一日中コードを書く人をカバーする、月額 200 ドルの「Max 20x」プランを利用しています。しかし、(開発者だけでなく) 非同期作業に Claude のようなエージェントを使用する人が増えており、これが Anthropic からの大きなポリシーと価格の変更を引き起こしました。
OpenClaw は、Claude のようなエージェントを対話型ではなく非同期で実行することを普及させました。マーケティング、販売、サポート、コーディングのタスクを自動化する新しくて強力な方法だったため、急速に成長しました。これらのタスクは 24 時間年中無休で実行されるため、簡単に大量のトークンを焼き付けることができます。
非同期により誰もがヘビーユーザーになる
Anthropic の計画は、ほとんどの Claude ユーザーが制限よりも大幅に少ない使用量を使用するため、経済的に合理的です。そのため、ライトユーザーはヘビーユーザーに補助金を出します。多くの Claude インスタンス (GitHub Actions、OpenClaw など) を非同期的に実行しているライト ユーザーがヘビー ユーザーになることがよくあります。
ヘビーユーザーは API レートで請求されるようになりました
2026 年 6 月 15 日以降、Claude を非同期的に使用すると、新しい毎月の「エージェント SDK」クレジットが引き出されます。クレジットは最大 5 回の場合は月額 $100、最大 20 回の場合は $200 です。 Agent SDK クレジットを超えると、使用量は標準 API レートで請求され、すべてのヘビー ユーザーに影響します。
ヘビーユーザーはCodexに移行するだろう
OpenAI が Codex サブスクリプションに対して同期エージェントと非同期エージェントの作業を同等に扱い続ける場合、これはすぐに上級ユーザーのデフォルトの選択肢となるでしょう。その結果、コーデックスは深刻な市場シェアを奪う可能性がある。
人間はオウンゴールを回避できる
クロードは今でも間違いなく最高のコーディング エージェントであり、まもなくほとんどのエージェントの作業が非同期になるでしょう。 Anthropic は、この価格変更を元に戻すか、非同期エージェント向けに構築された新しいサブスクリプション プランを導入することで、これを受け入れることができます。

Anthropic's new Agent SDK pricing is a win for Codex
Claude subscriptions were designed for developers
Many devs use the $200/mo "Max 20x" plan, which covers someone writing code all day with agents. But more and more people (not just devs) are using agents like Claude for async work, which triggered a major policy and pricing change from Anthropic.
OpenClaw popularized running agents like Claude asynchronously instead of interactively. It grew quickly because it was a new and powerful way to automate marketing, sales, support, and coding tasks. Because they run 24/7, these tasks can easily burn a lot of tokens.
Async makes everyone a heavy user
Anthropic's plans make financial sense because most Claude users use significantly less than their limits, so light users subsidize heavy ones. Light users running a number of Claude instances asynchronously (GitHub Actions, OpenClaw, etc) often become heavy users.
Heavy users now get billed at API rates
Starting June 15, 2026, using Claude asynchronously will draw from a new monthly "Agent SDK" credit. The credit is $100/mo on Max 5x, $200 on Max 20x. Beyond Agent SDK credit, usage is billed at standard API rates, which will impact all heavy users.
Heavy users will move to Codex
If OpenAI continues to treat synchronous and asynchronous agent work equally for Codex subscriptions, it will quickly become the default choice for advanced users. Codex could take serious market share as a result.
Anthropic can avoid an own-goal
Claude is still arguably the best coding agent, and soon most agentic work will be async. Anthropic could embrace that by reverting this pricing change or by introducing new subscription plans built for async agents.
