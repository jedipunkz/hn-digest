## Original Extract

Most of my professional work is in embedded systems, typically using C++. I've been doing more of my hobby work in Python (MicroPython/CircuitPython) and was thinking of investigating Rust. But then it occurred to me that if we're using AI more and more to generate code, does it really matter what language is used? If an agent is writing most of my code from prompts, I just need to understand the language well enough to verify that the reviewing agent is doing a good enough job and maybe tweak it a bit when I have to. If we can trust the AI to avoid the common problems, then what difference does the language make?
