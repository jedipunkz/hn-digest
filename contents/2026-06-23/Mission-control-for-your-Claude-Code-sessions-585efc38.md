Mission control for your terminal — a macOS menubar dashboard that mirrors what every Ghostty tab is doing (shell commands, dev servers, Claude Code sessions) and jumps you back to the right tab.

Requirements: macOS 14+, Ghostty, zsh, jq, and Claude Code (for agent rows).
