HN テキスト: どの AI ゲートウェイがうまく機能しますか?現在、AI エージェント (val.town 上) に Anthropic のルーターを直接使用していますが、OpenRouter や Vercel AI Gateway、あるいはまったく別のものを追加することを検討しています。私はゲートウェイについて、新しい VS Code フォークであるようだということ以外はほとんど知りません (つまり、誰もがそれを行っています)。私のざっとした調査によると、OpenRouter には最も多くのモデル/プロバイダーがありますが、5.5% の従量課金制のプラットフォーム料金が請求され (推論コストは顧客に転嫁されます)、プロンプト キャッシュなどの機能はサポートされていません。私たちは主に Cloudflare ショップであり、彼らも AI ゲートウェイを持っていることを数分前に知りました。しかし、これまでのところ、私が見つけた「ベスト LLM ゲートウェイ 2026」タイプの記事は...ゲートウェイを備えた企業によって公開されています。

## Original Extract

What AI gateways work well? We currently use Anthropic's router directly for our AI agent (on val.town) but are considering adding OpenRouter or the Vercel AI Gateway, or something else entirely. I know little about gateways except that they seem to be the new VS Code forks (i.e. everyone is doing it). My cursory research tells me that OpenRouter has the most models/providers, but charges a 5.5% pay-as-you-go platform fee (we pass-through inference costs to customers) and doesn't support features like prompt caching. We're mostly a Cloudflare shop, and I learned minutes ago that they, too, have an AI gateway--but so far the "Best LLM Gateway 2026" type articles I've found are published by...companies with a gateway
