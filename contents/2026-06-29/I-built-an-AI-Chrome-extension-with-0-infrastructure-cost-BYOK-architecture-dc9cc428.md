Professional pull request review inbox with analytics, actions, and integrations.
PR Focus Pro turns your GitHub PR inbox into a prioritized, summarized, and actionable queue — so you review what matters first, not what arrived first.
❓ The problem: GitHub shows PRs in the order they were opened. A one-line typo fix and a PR touching authentication code get the exact same visual weight. You spend more time deciding what to review than actually reviewing.
✅ PR Focus solves this: AI-powered risk scoring, plain-English summaries, and one-click draft reviews. All from your browser. All local. No backend.
---
✨ FEATURES
🔍 AI-POWERED TRIAGE
Each PR gets a 0–100 risk score based on CI status, PR age, and code scope (auth/DB/infra). AI summaries generated from the actual diff — not the title.
⚡ ONE-CLICK DRAFT REVIEWS
Generate approve or request-changes drafts directly from the diff. Edit and send without leaving the extension.
🧠 HYBRID PRIORITY QUEUE
Combines CI failures, PR age, and AI risk into a single sort. Failing CI always floats to the top.
👥 MULTI-ACCOUNT GITHUB
Switch between personal and work accounts instantly. No token juggling.
📊 ANALYTICS DASHBOARD
Weekly activity chart, KPIs, and risk breakdown (high/medium/low). Track your review velocity.
🔔 STALE PR NOTIFICATIONS
Get alerts after X days without review. Optional Slack/Discord webhook with risk score included.
📎 EXPORT TO CSV / PDF
Generate reports in seconds for billing or audits.
🌓 DARK / LIGHT / SYSTEM THEMES
The UI adapts to your preference — dark mode done right.
🔒 100% LOCAL — BYOK
Bring your own OpenAI, Groq, Mistral, or Ollama key. Your code never touches our servers. Zero telemetry. No tracking. Your GitHub token and AI key are stored locally in chrome.storage.local — never transmitted anywhere except directly to GitHub and your chosen AI provider.
