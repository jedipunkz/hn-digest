Enter a business question in plain language (or speak it) and the app turns it into a safe read‑only SQL query that runs on a synthetic SaaS database. It shows the generated SQL and displays the re...

I built this and it is open source. You ask a business question in plain English and an LLM (Llama 3.3 70B via Groq) turns it into SQL and runs it against a sample SaaS database — read-only, SELECT only. It shows the exact SQL it generated. Code: https://github.com/Chupacharcos/voice-to-sql-dashboard
More demos: https://adrianmoreno-dev.com Happy to take feedback.
