思い起こせば 2022 年に、私たちは Web サイト「EdgeOS」について報告していました。

Edge 上に構築されたオペレーティング システムです。しかし、それは本物ではなく、でっちあげだったことが判明しました。しかし当時私たちは、ブラウザが急速に肥大化していたため、Microsoft はおそらくこれが素晴らしいアイデアだと考えるだろうと述べていました。
興味深いことに、2年後の2024年には、Microsoftは実際にそのようなもの、つまり「Edge上に構築され」「AIエージェントによって駆動される」実際の軽量Windows OSに取り組んでいたようです。 「Project Aion」と名付けられたこのプロジェクトは、Copilot と Edge のみをベースにしていたでしょう。本日、この画像とビデオが BetaWiki discord で公開されました。この時期、AI を搭載したクラウド ベースの Windows 12 について多くの話題があったのも不思議ではありませんが、その一部は Microsoft 自体によって煽られ、Panos Panay のような大物幹部が Windows 12 を宣伝しました。
流出したビデオのナレーターは、「Aion は、Copilot をシェルのコアにネイティブに組み込む Web ベースのエージェント OS の一例です」と述べています。そういえば、Aion の一般的なシェル ユーザー インターフェイス (UI) は Windows 11 によく似ていますが、いくつかの大きな違いがあります。おそらく最も注目すべき点は、[スタート] ボタンが Copilot キーに置き換えられていることです。
考えてみれば、なぜ Microsoft が 2023 年に AI と Copilot の開発にこれほど熱心に取り組んでいたのか、今ならすべて理解できます。当時 CEO の Satya Nadella は、新たに導入された Copilot キーを革命的なスタート ボタンの再来のようにさえ例えていました。
内部では、Windows Aion は Win3 上で実行されていることが示されています。Windows Central によると、Win32 は Win32 レガシー アプリのサポートのない最小限の Windows コードベースでした。画像とビデオは、Windows だけでなく AOSP でも動作することを示唆しているように見えます。これは、同社が最近発表した Project Solara に非常に似ているように思えます。そのため、おそらく Aion は、Micros の変革を支援するために、Solara に進化しただけなのかもしれません。

Back in 2022 if you recall, we had reported on "EdgeOS", a web-powered operating system built on Edge. Turns out though it was not real and was a hoax; however at the time we had remarked on how Microsoft would probably think it's a great idea since the browser was getting more and more bloated fairly rapidly.
Interestingly two years down the line in 2024, it seems Microsoft was indeed working on such a thing, an actual lightweight Windows OS "built on Edge" and "driven by AI agents". Dubbed "Project Aion", it would have been solely based around Copilot and Edge. Images and videos of this surfaced today on BetaWiki discord. No wonder there was a lot of buzz about alleged AI-powered and cloud-based Windows 12 during this time, and some of it was fueled by Microsoft itself with big execs like Panos Panay hyping it up .
The narrator in the leaked video says "Aion is an example of a web-based agent OS that natively builds Copilot into the core of the shell." Speaking of which, while the general shell user interface (UI) of Aion looks quite similar to Windows 11, there are some major differences. The most noticeable one perhaps is that the Start button is replaced by the Copilot key.
Come to think of it, this all makes sense now as to why Microsoft was pushing on the AI and Copilot gears so hard back in 2023, as at the time CEO Satya Nadella had even compared the newly introduced Copilot key like the second coming of the revolutionary Start button .
Under the hood, Windows Aion is shown to be running on Win3 which, according to Windows Central, was a minimalist Windows codebase with no Win32 legacy app support. The images and video seem to suggest it would work on AOSP as well as Windows, which sounds quite similar to the company's recently announced Project Solara , so perhaps Aion has simply evolved into Solara so as to help turn Microsoft's agentic Windows dreams into a reality.
Bear in mind though that it's not confirmed that Microsoft was ever indeed working on such a thing and as such, this could certainly be a very well-made hoax too, similar to the earlier EdgeOS thing.
