Anthropic からの返答は見つからず、同社はこの問題について公の場で完全な沈黙を守り、私の通信にも返答しませんでした。その沈黙だからこそ、私は今、その後の正式なステップを踏まざるを得なくなっているのです。
Claude Desktop によるネイティブ メッセージング ブリッジのインストールに関連する問題に関して Anthropic が連絡を取っていないことを考慮して、できる限り透明性を保つために、私は、Anthropic に対して停止通知を発行せざるを得ませんでした。その内容は以下でご覧いただけます。 (トラフィックを考慮すると) これは公共の重要な関心事であることが明らかなので、問題が解決するまでブログを更新し続けるつもりです。

https://www.thatprivacyguy.com/blog/anthropic-spyware/
明確にしておきますが、これらのネイティブ メッセンジャー ブリッジの設置は、次のように EU 法の下で違法です。
指令 2002/58/EC 第 5 条 (3) には、要求されたサービスの提供に厳密に必要な場合を除き、情報の保存、またはエンド ユーザーの端末機器に既に保存されている情報へのアクセスは、エンド ユーザーに十分な情報が提供され、明示的な同意が得られた場合にのみ許可されると明記されています。
Claude Chrome をインストールしていないので (これらのマニフェストの目的です)、マニフェストを削除するアップデートを Claude Desktop (および Claude Code) にプッシュするようにする必要があります。これにより、ユーザーが Claude Desktop (または Claude Code) 用の Chrome プラグインをインストールするまでマニフェストを再度インストールすることはありません。私のマシンにクソを先制的にインストールし始めるのは問題ありません。

この事実は、単に eプライバシー指令の違反であるだけでなく、許可なくデータやソフトウェアをインストール、変更、消去、または破壊することを犯罪とする刑法第 337 条 C 条 (マルタ法第 9 章) の刑事違反でもあります。
私は反人類主義者ではありません (毎日貴社のサービスを利用しています) が、マルウェア/スパイウェアには反対であり、率直に言ってクソ行為には反対です。これはひどい動作だったので修正する必要があります。したがって、私はあなたにチャンスを与えます。これらのマニフェストを削除する新しいアップデートが 72 時間以内に公開されることを期待しています。それが実現しない場合、私は Anthropic に対して刑事および民事の告訴を提出し、そのことについては例外的に公表します。

To be clear, the installation of these Native Messenger bridges is illegal under EU law as follows:
Directive 2002/58/EC Article 5(3) clearly states that storage of information or gaining access to information already stored on the terminal equipment of an end user is only permitted if the end user is fully informed and has given their explicit consent - unless it is strictly necessary for the provision of the requested service.
