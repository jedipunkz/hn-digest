OpenAI のスタンドアロン デスクトップ ブラウザである ChatGPT Atlas は廃止され、新しい ChatGPT デスクトップ アプリが採用されます。
OpenAI は本日、まったく新しい ChatGPT デスクトップ アプリをリリースしました。これには、ChatGPT Codex とともに新しい ChatGPT Work エージェントが含まれています。
新しい ChatGPT デスクトップ アプリにより、OpenAI は堅牢なブラウザ機能を備えたデスクトップ アプリを提供できるようになりました。 ChatGPT (およびその前の Codex) も Chrome デスクトップ ブラウザ プラグインをサポートしています。これにより、Chrome ユーザーはブラウザを完全に切り替えることなく ChatGPT 統合の恩恵を受けることができます。
本日のリリースの一環として、OpenAI の James Sun は、スタンドアロンのデスクトップ ブラウザである ChatGPT Atlas が廃止されることを認めました。
最後に、これらすべてのアップデートをもちまして、Atlas を廃止する予定です。
これらすべての機能は、新しいブラウザーに飛びついた Atlas ユーザーから学んだことに基づいて構築されています。
あなたは、エージェントがオープン Web でのブラウジングと作業をより良くするためにどのように役立つかを教えてくれました。そして私たちは…
Sun 氏によると、「現時点での非推奨予定日は 8 月 9 日です。近日中にアプリ内と電子メールの両方で詳細情報を共有する予定です」とのことです。
ChatGPT Atlas は 10 月に Mac に登場しました。 OpenAI はその後、専用の Codex アプリをリリースし、4 月にアプリ内ブラウザー機能を追加し、今日新しい ChatGPT デスクトップ アプリ内にすべてをバンドルしました。

OpenAI is discontinuing ChatGPT Atlas, its standalone desktop browser

ChatGPT Atlas, OpenAI’s standalone desktop browser, is being sunset in favor of the new ChatGPT desktop app.
OpenAI released an all-new ChatGPT desktop app today, which includes a new ChatGPT Work agent alongside ChatGPT Codex.
With the new ChatGPT desktop app, OpenAI now has a desktop app with robust browser capabilities. ChatGPT (and Codex before it) also supports a Chrome desktop browser plugin. This lets Chrome users benefit from ChatGPT integration without completely switching browsers.
As part of today’s release, OpenAI’s James Sun confirms that ChatGPT Atlas, the standalone desktop browser, is being discontinued.
Lastly, with all these updates, we are going to be sunsetting Atlas.
All these capabilities were built on what we learned from Atlas users who took a leap of faith on a new browser.
You taught us how agents can help make browsing and doing work on the open web better, and we…
“The current targeted date for deprecation is 8/9, and we’ll share more information in the upcoming days both in-app and via email,” according to Sun.
ChatGPT Atlas arrived on the Mac back in October . OpenAI later released a dedicated Codex app, adding an in-app browser feature in April, before bundling it all inside the new ChatGPT desktop app today.
