## Original Extract

Been really loving this flow of having Claude Code spin up Codex agents for individual tasks. It keeps the context super clean while spreading the workload across both Claude and Codex subscriptions. So created this common CLI and the relevant skills, so Claude Code, Codex/ChatGPT (hosts) can delegate tasks to models (engines) powered by the claude cli or codex cli or both! Check it out and let me know what you think: https://github.com/muzam1l/coder
