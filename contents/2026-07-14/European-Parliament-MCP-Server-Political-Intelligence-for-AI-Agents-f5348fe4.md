🏛️ 欧州議会 MCP サーバー
欧州議会オープンデータ用のモデル コンテキスト プロトコル サーバー
🇪🇺 AI エージェント向けの EU 議会情報 · 🕵️ 15 個の OSINT 分析ツール · 🏛️ EP API v2 を完全にカバー · 🛡️ ISMS 準拠 · 🔒 GDPR 準拠の設計 · 📦 SLSA レベル 3
欧州議会オープン データ ポータルと MCP 対応 AI クライアント (Claude Desktop、VS Code、Cursor、GitHub Copilot) 間の正規 TypeScript MCP ブリッジ。 EU Parliament Monitor AI ニュースルームを強化しており、Hack23 政治情報ポートフォリオの Riksdagsmonitor の姉妹プロジェクトです。
欧州議会オープンデータ用のモデル コンテキスト プロトコル サーバー — 安全でタイプセーフな TypeScript 実装を通じて、AI アシスタントに議員、本会議、委員会、立法文書、議会質問への構造化されたアクセスを提供します。
MEP影響スコアリング（5次元モデル）、連合の結束力とストレス分析、政党の離反と異常の検出、グループ間比較分析、MEP/委員会の立法スコアリング、パイプラインのステータスとボトルネックの検出、委員会の仕事量と関与の分析、MEPの出席パターンと傾向、各国代表団の投票と構成、議会全体の政治情勢
📊 品質指標とドキュメント
欧州議会MCP

サーバーは Model Context Protocol (MCP) を実装し、AI アシスタント、IDE、およびその他の MCP クライアントに欧州議会のオープン データセットへの構造化されたアクセスを提供します。安全でタイプセーフな TypeScript/Node.js 実装を通じて、議員、本会議、委員会、立法文書、議会の質問に関する情報にアクセスします。
🔌 完全な MCP 実装: 63 ツール (9 コア + 3 アドバンス + 15 OSINT + 8 フェーズ 4 + 15 フェーズ 5 + 13 フィード)、9 リソース、および 7 プロンプト
🏛️ EP API v2 を完全にカバー : すべての欧州議会オープンデータ API エンドポイントをカバー
🕵️ OSINT インテリジェンス : MEP 影響スコアリング、連合分析、異常検出
🔒 セキュリティ第一 : ISMS 準拠、GDPR 対応、SLSA レベル 3 の来歴
🚀 高性能: 200ms 未満の API 応答、インテリジェントなキャッシュ、レート制限
📊 タイプ セーフティ : TypeScript 厳密モード + Zod ランタイム検証
🧪 十分なテスト済み : 80% 以上のコード カバレッジ、1130 以上の単体テスト、71 の E2E テスト ケース
📚 完全なドキュメント : アーキテクチャ、TypeDoc API (HTML + Markdown)、セキュリティ ガイドライン
プライマリコンパイラ: typescript-7 エイリアス経由の TypeScript 7 RC (node_modules/typescript-7/bin/tsc)
互換性レイヤー: @typescript/typescript6 は引き続き利用できるため、既存の 6.x ベースのツールは tsc6 と TypeScript 6 API を引き続き使用できます。
コアプラットフォーム: @modelcontextprotocol/sdk 1.29.0、zod 4.4.3、vitest 4.1.10
🌍 Hack23 政治情報エコシステム
この MCP サーバーは、AI が生成する政治インテリジェンスとリアルタイムの責任分析によって議会ジャーナリズムを破壊するという Hack23 の使命の EU データ バックボーンであり、公的立法データに適用される構造化オープンソース インテリジェンス (OSINT) のトレード クラフトを通じて民主的な透明性を高めます。
🎯 ビジョン: AI を活用した民主的な透明性
Hack23 AB ビルドがオープン-

すべての国民、ジャーナリスト、研究者、政策立案者の手に、厳密で証拠に基づいた政治分析を提供する情報源情報プラットフォーム。 MCP サーバー、エージェント AI ニュースルーム、オープンな議会データを組み合わせることで、以前は潤沢な資金を備えたロビー活動組織や社内の政策部門が生成する必要があった自動化されたインテリジェンス製品 (連合分析、投票パターンの解読、MEP/MP 影響スコアリング、立法パイプライン予測) を生成します。
「政治情報へのアクセスの民主化。かつては分析チームが必要であった作業が、AI アシスタントを使えば誰でもできるようになりました。」
このポートフォリオは無党派で、完全なオープンソース (Apache-2.0) であり、完全な ISO 27001:2022 / NIST CSF 2.0 / CIS Controls v8.1 に準拠した Hack23 ISMS に基づいて運用され、GDPR に準拠した設計であり、党派的な影響力のために兵器化されないようにアーキテクチャ的に設計されています。つまり、すべての政治団体の平等な扱い、公開データのみ、ユーザー アカウントなし、広告なし、追跡なしです。

The European Parliament MCP Server implements the Model Context Protocol (MCP) to provide AI assistants, IDEs, and other MCP clients with structured access to European Parliament open datasets. Access information about MEPs, plenary sessions, committees, legislative documents, and parliamentary questions through a secure, type-safe TypeScript/Node.js implementation.

🚀 High Performance : <200ms API responses, intelligent caching, rate limiting
