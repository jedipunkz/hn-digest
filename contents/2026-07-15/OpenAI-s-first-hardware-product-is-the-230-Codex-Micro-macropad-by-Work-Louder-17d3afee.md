OpenAI の Codex のユーザー数は間もなく 900 万人に達し、そのうちの少なくとも一部のユーザーは間もなく OpenAI のエージェント コーディング ツール、つまり Work Louder と提携して開発されたプログラム可能なメカニカル マクロパッド OpenAI を使用する新しい方法を手に入れることになります。
水曜日に数量限定で発売された230ドルのCodex Microは、Work Louderの薄型Creator Micro 2キーボードのバリエーションだ（同社は以前、Figmaと協力して同様のマクロパッドを発売しており、現在はFramerユーザー向けの特別版も提供している）。
在庫がなくなるまでご注文を受け付けております。
OpenAI初のハードウェア製品
定義の仕方にもよりますが、これは OpenAI の最初のハードウェア製品ですが、これは伝説的な Jony Ive とのコラボレーションではないことに注意してください。それは全く別の製品です。これは基本的に、以前の Work Louder キーボードのスキンを変更したバージョンです。
ハードウェア自体に関しては、Codex Micro は Work Louder 独自の薄型メカニカル スイッチを使用しているようですが、Work Louder が専門であることを考えると、これは驚くべきことではありません。

薄型キーボードに。
購入者はサイレントスイッチとクリック感のあるスイッチのどちらかを選択でき、Codex Micro を USB-C ケーブルまたは Bluetooth 経由で接続できます。
Creator Micro 2 と比較して、OpenAI バージョンは高価ですが、それには理由があります。
ダイヤルとジョイスティックに加えて、OpenAI と Work Louder が「エージェント キー」と呼ぶものを備えています。これらはパッド上の 6 つの上部キーで、エージェントの状態に応じて異なる色に光る光るキーキャップが付いています。 1 回タップするとキーに関連付けられたエージェントが選択され、2 回タップするとそのエージェントが最前面に表示され、色によって何が起こっているかが一目でわかります。白色のライトはエージェントがアイドル状態であることを意味し、青色は考え中であることを意味し、緑色は終了したことを意味し、赤色はエラーが発生したことを意味します。
Codex Micro 上のすべては Work Louder 独自の入力ツールを使用してカスタマイズできるため、ユーザーはこれら 6 つのエージェント キーが Codex に固定されたタスク、最新のタスク、または人間の入力が必要な優先タスクを表すかを選択できます。フルコントロールの場合、特定のエージェントをキーに割り当てるオプションもあります。
マクロパッドの下部には 6 つの通常のキーがあり、ユーザーが変更を承認または拒否し、スレッドを分岐できるようにデフォルトで割り当てられています。最下段の大きいボタンは、プッシュ トゥ トーク機能を有効にするように設定されています。シリコン バレーでは音声による Codex の使用が啓蒙の 3 番目のステップであるようです。
Codex Micro のエッジには LED が点灯するアクリルが付いているので、Codex が録音しているときに点灯するので、マイクがオンになっていることがわかります。
ジョイスティックに関しては、デフォルトでは Codex のコンポーザー バーを制御し、ダイヤルで選択を行うことができます。ただし、Codex Micro の他の機能と同様に、これらを他の機能にマッピングするのも簡単です。

キャップは交換可能です。 Codex Micro には、バイブコーディングを楽しむための「yolo」キャップや「yeet」キャップを含む 32 個の追加キーキャップが付属しています。
ここでは Codex に焦点を当てていますが、Work Louder の入力アプリを使用すると、最大 5 つの追加レイヤーを作成し、他のアプリケーションでも使用できます。

OpenAI’s Codex is about to hit 9 million users , and at least some of those users will soon get a new way of using OpenAI’s agentic coding tool: a programmable mechanical macropad OpenAI developed in partnership with Work Louder .
Launched on Wednesday in limited numbers, the $230 Codex Micro is a variation on Work Louder’s low-profile Creator Micro 2 keyboard (and the company previously launched a similar macropad in collaboration with Figma and currently offers a special edition for Framer users, too).
Orders are now open until supplies last.
OpenAI’s first hardware product
Depending on how you define things, this is OpenAI’s first hardware product, but it’s worth noting that this isn’t the fabled Jony Ive collaboration . That’s a completely different product . This is basically a reskinned version of a previous Work Louder keyboard.
As for the hardware itself, it looks like the Codex Micro is using Work Louders own low-profile mechanical switches, which is no surprise, given that Work Louder specializes in low-profile keyboards.
Buyers get a choice between silent and clicky switches, and can connect the Codex Micro with a USB-C cable or over Bluetooth.
Compared to the Creator Micro 2, the OpenAI version is more expensive, but there’s a reason.
In addition to a dial and joystick, it features what OpenAI and Work Louder call ‘agent keys.” These are the six top keys on the pad that have shine-through keycaps that light up in different colors depending on the state of your agents. A single tap selects the agent associated with the key, a double-tap brings it to the foreground, and the color tells you what is happening at a glance: a white light means the agent is idle, blue means it’s thinking, green means it’s finished, red means there was an error.
Since everything on the Codex Micro can be customized using Work Louder’s proprietary Input tool, users get to choose whether those six agent keys represent pinned tasks in Codex, the most recent tasks, or priority tasks that need human input. For full control, there is also the option to assign a specific agent to a key.
At the bottom of the macropad, there are six regular keys that, by default, are assigned to let the user accept or reject changes and branch threads. The larger button on the lowest row is set to activate the push-to-talk feature — because in Silicon Valley, using Codex by voice is seemingly the third step of enlightenment.
The Codex Micro has an acrylic edge that is also LED-lit, so when Codex is recording, it will light up, so you know the mic is on.
As for the joystick, by default, it controls the composer bar in Codex while the dial lets you make selections. But like with all things Codex Micro, it’s easy to map these to other features as well.
All of these keys can be remapped and the keycaps can be exchanged. The Codex Micro comes with 32 additional keycaps, including ‘yolo’ and ‘yeet’ caps for all your vibe-coding pleasure.
And while Codex is the focus here, Work Louder’s Input app lets you create up to five additional layers and use them to with any other application as well.
Do you need the Codex Micro? No.
Will it make you more productive? Unclear.
Is it worth $230? Only you can decide that.
