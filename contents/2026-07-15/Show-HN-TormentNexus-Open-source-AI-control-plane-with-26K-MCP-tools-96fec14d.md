TormentNexus は、LLM エージェントに無限の永続メモリ、ツール オーケストラを提供するように設計された、ローカル ファーストのオープンソース ユニバーサル AI コントロール プレーンおよびコーディング ハーネスです。

配給、自律的なインフラストラクチャの同化など、すべてマシン上または実稼働環境でローカルに実行されます。
🏢 企業およびエンタープライズ展開 (HyperNexus)
プロフェッショナル、企業、エンタープライズ展開については、HyperNexus にアクセスしてください。
HyperNexus は、正式にライセンスを取得し、認定され、権威のあるクラウドホスト型の企業向け TormentNexus カーネル インストールです。それは以下を提供します:
テナントの分離 : 自動化されたプロビジョニング スクリプトにより、分離されたボリュームとリソース制限を備えたテナントごとのコンテナーをスピンアップします。
エンタープライズ グレードのセキュリティ : 完全な SSO サポートと、きめ細かいロールベースのアクセス制御 (RBAC) ミドルウェア。
エアギャップと VPC のサポート : SIEM 統合のための毎日の監査ログを備えた、本番環境に対応した展開構成。
Stripe Billing Integration : 組み込みの Webhook 構成とサブスクリプション シート管理。
TormentNexus は、単なるアグリゲータではなく、意思決定システムおよびユニバーサル ブリッジです。これは、AI ツール、モデル、エージェントの混沌とし​​た状況を、AI 主導の開発のための単一の一貫したオペレーティング システムに統合するモジュラー モノリスとしてローカルで実行されます。

1. プログレッシブ MCP ツールのルーティングとパリティ
モデルは、50,000 トークンのツール ダンプで圧倒されるべきではありません。 TormentNexus は、多層の段階的な開示システムを採用しています。
セマンティック検索: ローカル ベクトル埋め込みは、無限サーバーのグローバル MCP ディレクトリに対してアクティブなプロンプトを照合します。
ルーター: 関連性の高い最上位のツール スキーマのみがアクティブな LLM コンテキストに挿入されます。
Universal Parity : Claude Code、Codex、Gemini CLI、Cursor、Windsurf のバイトごとに同一のツール シグネチャ。
2. デュアルティアメモリアーキテクチャ (L1 / L2)
コンテキストは有限です。メモリは無限でなければなりません。
L1

— セッション スクラッチパッド : アクティブなセッションに直接関連付けられた一時的な超高速メモリ。
L2 — The Vault : sqlite-vec を使用した SQLite の永続的なセマンティック ストレージ。正確なトランスクリプトと LLM 圧縮ヒューリスティックを保存します。
コンテキストハーベスティング : すべてのセッションが自律的に L2 ボールトにクエリを実行し、関連する履歴ヒューリスティックを取得します。
熱力学 : 関連性により熱が増加し、時間が経過すると減衰が引き起こされます – 生物学的記憶モデリング。
3. 回復力のある LLM ウォーターフォール

Core Pillars
1. Progressive MCP Tool Routing & Parity
Models should never be overwhelmed with a 50,000-token tool dump. TormentNexus employs a multi-layered, progressive disclosure system:
Semantic Search : Local vector embeddings match the active prompt against a global MCP directory of infinite servers.
The Router : Only the top highly relevant tool schemas are injected into the active LLM context.
Universal Parity : Byte-for-byte identical tool signatures for Claude Code, Codex, Gemini CLI, Cursor, and Windsurf.
2. Dual-Tier Memory Architecture (L1 / L2)
Context is finite; memory must be infinite.
L1 — Session Scratchpad : Ephemeral, lightning-fast memory tied directly to the active session.
L2 — The Vault : Permanent semantic storage in SQLite with sqlite-vec . Saves exact transcripts and LLM-compressed heuristics.
Context Harvesting : Every session autonomously queries the L2 Vault to pull in relevant historical heuristics.
Heat Mechanics : Relevance increases heat, time causes decay — biological memory modeling.
3. The Resilient LLM Waterfall
Uptime is non-negotiable. TormentNexus's inference client natively catches 429s (Rate Limits) and 5xx (Server Errors), seamlessly cascading the exact payload down a prioritized chain without crashing:
OpenRouter (Secondary aggregator fallback)
Local LM Studio / Ollama (Ultimate offline fallback)
