## Original Extract

Claude *Code* to me it the real revolution (not web chat), and all my colleagues are on it. I use it outside my job (sysadmin) a lot, and trying to push my non-IT friends to use it too. To me it is so clear that to use Claude *Code* you need to know the basic of bash... so I really see a huge barrier for my non-IT friends on it. I tried to teach them, but it's clearly a big wall for them: most of them have Mac, and they need to know the basic (e.g. "cd "): even a proper cut/paste of a bash command in Terminal is not that simple. Then we had to give CC and API token by using env variable... a nightmare for them. Many give up and go back to AI web chat that they thing is really limited (since it tells them what to do, but does not do nothing itself)
