BatchEdits MCP サーバーとは何ですか?
BatchEdits MCP (Model Context Protocol) サーバーは、BatchEdits を Claude、Cursor、Windsurf、ChatGPT などの AI エージェントに接続します。接続すると、AI エージェントはビデオのアップロード、無音部分の削除、キャプションの追加、完成したビデオのエクスポートを行うことができます。これらはすべて、AI ワークスペース内の 1 つのテキスト プロンプトからトリガーされます。

BatchEdits MCP サーバー URL (https://batchedits.com/api/mcp) を Claude または Cursor MCP 構成に追加します。接続したら、AI ワークスペースを離れることなく、AI エージェントにビデオのアップロード、無音部分の削除の適用、キャプションの生成、完成したファイルのダウンロードを指示します。
